## 📖 About the Project

**ManaChain** is a community platform that connects brands with their supporters through a unique engagement system. By holding brand tokens, community members gain access to exclusive privileges, special events, and unique experiences with the brands they believe in.

The platform empowers emerging brands to build and grow their communities from the ground up, while giving supporters a meaningful way to engage with and support the brands they love.

### Main Features

- **Brand Discovery** : Swipe interface (Tinder-style) to discover and support emerging brands
- **Community Engagement** : Hold brand tokens to unlock exclusive privileges and access to special events
- **Brand Application** : Complete registration process allowing new brands to join the platform and build their community
- **Emerging Brands Support** : Platform designed to help young brands emerge and connect with their early supporters
- **Adaptive Theme** : Interface with light/dark mode

## 🛠️ Tech Stack

- **Frontend** : Next.js 16, React, TypeScript, Tailwind CSS
- **Backend** : Node.js, Express, TypeScript
- **Database** : PostgreSQL (Supabase)
- **Web3** : Dynamic SDK, Wagmi, Viem
- **Storage** : Pinata IPFS

## 🚀 Quick Start

### Prerequisites

- Node.js 18+
- pnpm
- PostgreSQL (or Supabase account)

### Installation

```bash
# Install dependencies
pnpm install

# Configure environment variables
cp .env.example .env
# Edit .env with your API keys

# Start the backend server
cd server
pnpm start

# Start the client (in another terminal)
cd client
pnpm dev
```
